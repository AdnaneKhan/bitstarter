Whether you are a seasoned programmer, college student, or learning to code in your free time, there is a role for you to play in the Coder's Quest.
Coder's Quest isn't about completing pre-scripted programming exercises, it's about completing real programming tasks that contribute to developing real web-based applications.
Coder's quest can be thought of as a hybrid between open source development and multiplayer role-playing games. Wait...what?
Coder's Quest matches you with 2-4 other users and sets that team on an objective. While it is small and intended to be completed in one sitting, your objective isn't just a game or exercise, it's a component of an actual open source project that other groups are also working on completing, it could be anything from fixing a bug to adding functionality to a video game character.

Why support Coder's Quest?
Coder's Quest will be focused on being designed with open source as its core philosophy. Everything done on Coder's Quest is giving back to its supporters and the internet community.
You can donate Bitcoins or can contribute your time by helping with programming, graphic design, or spreading the word via social networking, any contribution will be appreciated and put this effort closer to realization.

FAQ

When will Coder's Quest be released?
Coder's Quest will be released incrementally, anyone who supports by donating any amount, providing their skills, or sharing on twitter will be provided alpha access as soon as the servers have been set up.
What will Coder's Quest run on?
Coder's Quest will initially be released to run completely within a web browser that supports HTML5.
A native app. for Android Tablets will come soon after.
I want to help, how can I?
If you can program, design graphics, or write good stories to accompany the Coding Quests, then your help can be used. Please send an email and look at our git repository for up to date development tasks.
What languages will Coding Quests be in?
We plan to support Python and Javascript with Node.js as our initial languages, Cloud9 will be used as our backend for the web based IDE.
